# 📊백준 문제풀이 현황 확인 프로그램💻
유저의 백준 닉네임을 입력하면 등급별 푼 문제의 개수와 경험치를 그래프로 화면에 띄워주는 프로그램입니다. [solved.ac](https://solved.ac/)의 API를 사용하였습니다.
